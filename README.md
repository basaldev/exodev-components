# devhub-components
React UI Components for DevHub

## Get started

```
npm ci
```

## Development

### Start dev with storybook

```
npm start
```

## Lint

```
npm run lint
```

## Test

```
npm test
```

A test by [Jest with ts-jest](https://basarat.gitbooks.io/typescript/docs/testing/jest.html)

```
npm test watch
```

For live test

## Build

```
npm run build
```
